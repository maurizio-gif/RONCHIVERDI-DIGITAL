06 — Identity Resolution & Tracking visite
Come funziona il browser_id
1
Prima visita — anonima
Il sito genera un UUID e lo salva in localStorage. Ogni page view viene inviata a Supabase con questo ID anonimo. L’utente è ancora sconosciuto.
2
Compilazione form — identità risolta
Il browser_id viene aggiunto come campo nascosto al form. n8n lo riceve insieme ai dati, crea il contatto e salva il browser_id nel profilo. Da questo momento tutte le visite passate e future sono attribuite a quella persona.
3
Visite successive — utente riconosciuto
Il sito legge il browser_id dal localStorage, lo invia con ogni page view. Supabase sa chi è, quante volte è tornato, quali pagine ha visitato e da quanto tempo non si vede.
4
Cross-device — magic link
Se l’utente torna da un nuovo dispositivo è di nuovo anonimo. La soluzione: nell’email di conferma form si include un link tracciato. Cliccandolo, il nuovo browser_id viene automaticamente collegato al contatto già esistente.

JS — da includere nel <head> del sito tracker.js 
// Genera o legge il browser_id
function getBrowserId() {
  let id = localStorage.getItem('rvid');
  if (!id) {
    id = crypto.randomUUID();
    localStorage.setItem('rvid', id);
  }
  return id;
}

// Traccia ogni page view
function trackPageView() {
  fetch('https://<supabase>/rest/v1/page_views', {
    method: 'POST',
    headers: { 'apikey': '<anon_key>', 'Content-Type': 'application/json' },
    body: JSON.stringify({
      browser_id: getBrowserId(),
      url: location.href,
      referrer: document.referrer
    })
  });
}

// Inietta il browser_id nei form
document.querySelectorAll('form').forEach(f => {
  const h = document.createElement('input');
  h.type = 'hidden'; h.name = 'browser_id';
  h.value = getBrowserId();
  f.appendChild(h);
});

trackPageView();

OVERRIDE COMMERCIALE — snooze
Se opportunità.snooze_fino_al > today, il lead viene escluso dalla lista lavoro anche se lo score è alto. Il commerciale usa questo campo quando il lead ha detto “richiamami tra X giorni”. n8n alla mezzanotte rimuove automaticamente lo snooze scaduto e notifica il commerciale assegnato.
